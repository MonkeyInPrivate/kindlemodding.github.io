# Setting Up Custom Screensavers
Now that you have jailbroken your Kindle, it is possible to replace the default Amazon screensavers with custom ones of your chosing.

<p class="important">You must <a href="./koreader">Install KOReader</a> before you can set up custom screensavers.</p>
